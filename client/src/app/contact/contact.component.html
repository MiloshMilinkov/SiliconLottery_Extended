Silicon Lottery Project

The "Silicon Lottery" initiative constitutes a graduate thesis under the auspices of the course "Integration of Software Technologies." Fundamentally, the "Silicon Lottery" encompasses the development of an e-commerce web application, designed to emulate the functionalities of an online retail platform specializing in the sale of computer hardware.

Technological Frameworks

This project will be executed utilizing a triad of technological frameworks:

  • AngularJs
  • .NET Web API (Version 7.0.3)
  • Database Management Systems: Microsoft SQL (Server Version 18, for Production use) and SQLite (for Development purposes)

Personal Challenges

Additionally, the undertaking will involve personal challenges, including comprehensive version control using GIT and the complete development of the application within the Visual Studio Code Integrated Development Environment (IDE).
